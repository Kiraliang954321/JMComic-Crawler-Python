# JM Comic 下载插件

在 QQ 私聊或群聊中发送 JM 码，自动下载整本漫画并打包 ZIP 发送。

## 功能

- 支持 JM 码格式：`JM123456`、纯数字 `123456`、JM URL
- 全部章节自动打包为一个 ZIP
- 后台任务队列，支持多人使用
- 群聊需 @机器人

## 命令

| 触发方式 | 说明 |
|----------|------|
| 发送 JM 码 | 加入下载队列 |
| `/jm_status <任务号>` | 查询下载进度 |

## 配置

在 AstrBot 管理面板中配置：

- **下载目录** / **ZIP 输出目录** — 文件存储位置
- **图片并发数** / **章节并发数** — 下载速度控制
- **JM API 域名** — 留空使用内置域名
- **代理 / Cookie** — 网络配置

## 依赖

需要 `jmcomic` 库。两种方式安装：

1. `pip install jmcomic`（推荐）
2. 或在配置中指定 `jm_lib_path` 指向 JMComic-Crawler-Python 项目根目录
